## Bem-vindo à API StarApp

A **API StarApp** foi desenvolvida para permitir que terceiros integrem seus estabelecimentos ao sistema **AppBarber** e **AppBeleza**. A API facilita a gestão de agendamentos e consultas de serviços/profissionais disponíveis, além de outras funcionalidades essenciais.

A documentação cobre os seguintes tópicos:

- **Autenticação de Usuários** (Login, Cadastro, Recuperação de Senha)
- **Gestão de Serviços** (Consulta de serviços disponíveis, profissionais, horários)
- **Agendamento de Serviços** (Realização e cancelamento de agendamentos)

## Autenticação

Todos os endpoints da API são protegidos por tokens, exigindo autenticação em cada requisição.
