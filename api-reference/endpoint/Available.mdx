## Available Schedules

Este endpoint permite consultar os horários disponíveis para agendamento de um serviço em um determinado período.

### Método HTTP

`GET`



<api-playground method="GET" url="http://localhost:4000/api/available"/>


## Text Formatting

| Header     | Tipo   | Descrição                                                  | Obrigatório |
|-----------------|--------|------------------------------------------------------------|-------------|
| `client-id`     | string | ID único do cliente que está requisitando                   | Sim         |
| `client-code`   | string | Código do cliente requisitante                              | Sim         |
| `device-id`     | string | Código do dispositivo do qual o cliente está requisitando   | Sim         |
| `authorization` | string | Chave de autenticação da API no formato Basic               | Sim         |


| Parâmetro    | Tipo    | Descrição                                                                                         | Obrigatório |
|--------------|---------|---------------------------------------------------------------------------------------------------|-------------|
| `start_date` | string  | Data de início no formato `DD/MM/YYYY HH:mm:ss` para consulta de horários disponíveis              | Sim         |
| `end_date`   | string  | Data de término no formato `DD/MM/YYYY HH:mm:ss` para consulta de horários disponíveis             | Sim         |
| `service`    | string  | ID do serviço a ser consultado                                                                    | Sim         |
| `professional`| string | ID do profissional (opcional)                                                                     | Não         |
| `combo`      | string  | ID do combo, obrigatório quando o serviço for `0`                                                  | Condicional |
