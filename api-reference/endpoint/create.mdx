## Available Schedules

Este endpoint permite consultar os horários disponíveis para agendamento de um serviço em um determinado período.

<api-playground method="GET" url="{{base_url}}/api/available">

  ### Authorization
  <api-header name="authorization" type="string" value="Basic {{api_client_key}}" required>
    Authorization string (Basic)
  </api-header>

  ### Headers
  <api-header name="client-id" type="string" required>
    ID do cliente requisitante
  </api-header>
  <api-header name="client-code" type="string" required>
    Código do cliente requisitante
  </api-header>
  <api-header name="device-id" type="string" required>
    ID do dispositivo do cliente
  </api-header>

  ### Query Params
  <api-param name="start_date" type="string" required>
    Data de início no formato `DD/MM/YYYY HH:mm:ss`
  </api-param>
  <api-param name="end_date" type="string" required>
    Data de término no formato `DD/MM/YYYY HH:mm:ss`
  </api-param>
  <api-param name="service" type="string" required>
    ID do serviço
  </api-param>
  <api-param name="combo" type="string" optional>
    ID do combo (obrigatório se `service` for 0)
  </api-param>

</api-playground>

### Método HTTP

`GET`

{{base_url}}/api/available
